# MYnyak Engsel Sunset

CLI client for a certain Indonesian mobile internet service provider.

# How to get environtment Variables
Go to [OUR TELEGRAM CHANNEL](https://t.me/)
Copy the provided environment variables and paste it into a text file named `.env` in the same directory as `main.py`.
You can use nano or any text editor to create the file.

# How to run with TERMUX
1. Update & Upgrade Termux
```
pkg update && pkg upgrade -y
```
2. Install Git
```
pkg install git -y
```
3. Clone this repo
```
git clone https://github.com/kcepu877/xr
```
4. Open the folder
```
cd xr
```
5. Setup
```
bash setup.sh
```
6. Run the script
```
python main.py
```
7. Auto Install to vps
```
wget https://raw.githubusercontent.com/kcepu877/XR/refs/heads/main/v2.sh && chmod +x v2.sh && ./v2.sh

```
8. Auto Install to TERMUX
```
wget https://raw.githubusercontent.com/kcepu877/XR/refs/heads/main/v1.sh && chmod +x v1.sh && ./v1.sh

```
